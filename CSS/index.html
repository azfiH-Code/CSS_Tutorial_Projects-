CSS Complete Tutorial

This page covers all important CSS properties, their types, and live examples.

CSS Property Types

CSS properties can be grouped into types:

  • Text & Font → font-family, font-size, font-weight, text-align, text-decoration, text-transform
  • Box Model → margin, padding, border, width, height
  • Background & Color → background-color, background-image, color, opacity
  • Display & Visibility → display, visibility, overflow
  • Positioning → position, top, left, z-index
  • Flex & Grid Layout → display:flex/grid, justify-content, align-items, grid-template-columns
  • Transitions & Animations → transition, transform, @keyframes, animation
  • Pseudo-classes & Pseudo-elements → :hover, :focus, :nth-child(), ::before, ::after
  • CSS Variables → --variable-name

CSS Animation (@keyframes)

Animated Box

@keyframes move {
    0% { transform: translateX(0); }
    50% { transform: translateX(100px); }
    100% { transform: translateX(0); }
}
.animation-demo {
    width: 100px;
    height: 100px;
    background-color: #ff5733;
    animation: move 3s infinite;
}
